$SCI2WEB has a complete documentation intended for Application Developers, Final Users and Web Masters.

Available on-line documentation includes:

  • $SCI2WEB server site installation (web masters).
  • Porting a scientific application to a $SCI2WEB server site (developers).
  • Using a Sci2Web ported application (users and developers).
  • Searching for results in the $SCI2WEB database for a given application (visitors).

Other documents available here are:

  • Sci2Web white paper (web masters and developers).

If you want to use a reference to $SCI2WEB in a paper or a technical document please cite the following paper:

Zuluaga, J.I. Sci2Web: bringing naturally developed scientific applications to the web, in preparation, 2011.

More documentation and other related material and documents can be found at the Sci2Web home site.
